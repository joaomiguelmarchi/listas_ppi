Diferença entre Web e Internet

Internet: é uma rede global de computadores conectados que permite a troca de informações. Ela inclui diversos serviços como e-mail, FTP e, claro, a Web.

Web: é um serviço específico da Internet. Utiliza o protocolo HTTP para acessar documentos, como páginas HTML, que são visualizadas por meio de navegadores.
